# AdGenius AI 🚀

AI-powered UGC Advertisement Generator built using Prompt Engineering, React, Tailwind CSS, and TypeScript.

## 📌 Project Overview

AdGenius AI helps marketers, startups, and local businesses generate:

* Viral Hooks
* UGC Ad Scripts
* Instagram Reel Captions
* CTA Lines
* Influencer Style Ads

using AI-powered prompt engineering.

## ✨ Features

* AI Prompt Studio
* UGC Script Generator
* Viral Hook Generator
* CTA Generator
* Modern SaaS UI
* Responsive Design
* Dashboard Analytics
* Dark/Light Mode
